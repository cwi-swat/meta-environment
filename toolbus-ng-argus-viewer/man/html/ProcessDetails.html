Process Details


When more information about a process is needed, the Process Details view can be used. Here you find an overview of the following available details:

  • Subscriptions
  • Process State
  • Process Note Queue
  • Process Variables
    •

    The Process Details can be viewed by clicking the Process Details tab after selecting a process from the Process list. The available details will be described below in more detail.

    Subscriptions


    Under subscriptions you can see for what messages the selected process has subscribed.

    Process State


    Under state you can see what statements have been executed by the selected process.

    Process Note Queue


    Under Note Queue you can see which notes are in the note queue of the selected process.

    Process Variables


    Under Variables you can view the variables used by the selected process and their current value. To display or refresh these variables, you need to click on a statement in the state window.
