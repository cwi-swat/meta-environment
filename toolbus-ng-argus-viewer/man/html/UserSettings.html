User Settings


When you want to set the User Settings, click on Applications and then Settings. Here you can set the following properties:

  • History limit
  • Automatic statement focus
  • Collapse whitespace
    •

    History limit


    History limit represents the total steps that you can see back in time.

    Automatic statement focus


    When a step is executed the focus will change. This could be obscure and unwanted. This can be solved when checking this feature.

    Collapse whitespace


    When there are processes invisible and these processes contains many statements there exists much whitespace what can cause an obscure visualisation. This can be solved when checking this feature.
