Architecture Performance View


When more information is needed about the architecture or performance, click on Architecture View. The Architecture View holds the following information:

  • The global archtecture of the script
  • The performance of the elements
    •

    The global archtecture of the script


    The global architecture is generated while running the script. It displays tools and processes and allows you to see how they communicate. It also indicates the performance of the tools and processes. Good performing parts are green, yellow and orange is a bit worse, red is bad. Tools that do not have any performance information available are displayed in black.

    The performance of the elements


    The details of the performance are displayed in the table below the archtecture display. It allows you to recieve more detailed information about the tools and processes that you can select in the architecture display.
