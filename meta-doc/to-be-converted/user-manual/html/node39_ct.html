Defining Strings

Figure 14 The specification below[]CODE:string gives the lexical definition of strings which may contain escaped double quote characters. It defines a StringChar as either

  • zero or more arbitrary characters except double quote or newline, or
  • an escaped double quote, i.e., \".

A string consists of zero or more StringChars surrounded by double quotes.
