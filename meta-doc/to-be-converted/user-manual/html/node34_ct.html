Option

If zero or exactly one occurrence of a lexical token is desired the option operator described in Option[ Section ]OptionOperator can be used.

The use of the option operator is illustrated below[ in Figure ]CODE:option. Identifiers are defined consisting of one letter followed by one, optional, digit. This definition accepts a and z8, but rejects ab or z789.
