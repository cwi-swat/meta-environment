Lexical constructor functions

For each production in a lexical syntax section, a lexical construction function is automatically added. Lexical construction functions are used to construct and deconstruct lexical syntax using patterns.

For each production A B C -> D in a lexical syntax section, the generated constructor function is of the form: d "(" A B C ")" -> D. Here, ``d'' is the name of the sort `D` written in lowercase letters only.

The lexical constructor functions will be discussed in more detail in the section on equations[ see Section ]Equations. This part of the manual is severely different from previous versions, since before Asf+Sdfhad flat lexical constructors and since version 2.0 it has fully structured lexical constructors.
