Sdf comment convention

The comment convention within an Sdf specification is that character between "%%" and the end of line is comment as well as every character between two "%" including the newline character. An example of the use of comments is given below[ in Figure ]CODE:sdfcomment.

Figure 3: Sdf comment

This definition also defines the comment convention in Sdf itself. More details on defining layout can be found in Restrictions[Section ]Restrictions.
