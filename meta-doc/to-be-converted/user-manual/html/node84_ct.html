Default Equations

The evaluation strategy for normalizing terms given the equations is based on innermost rewriting. All equations have the same priority. Given the outermost function symbol of a redex the set of equations with this outermost function symbol in the left-hand side is selected and all these rules will be tried. However, sometimes a specification writer would like to write down a rule with a special status ``try this rule if all other rules fail''. A kind of default behaviour is needed. Asf offers functionality in order to obtain this behaviour. If the of an equation starts with default- this equation is considered to be a special equation which will only be applied if no other rule matches.
