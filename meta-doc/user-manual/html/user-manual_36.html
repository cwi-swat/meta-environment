7 Selected Components of the ASF+SDF Meta-Environment

Now we will describe a selection of components that are useful for building stand-alone tools. Be aware that the information in the following pages is still very much in a state of flux.

Note that all components mentioned have a Unix-style manual page, e.g., man sglr gives the manual page for the command sglr. All components also have a -h options that gives an overview of their command line options.
