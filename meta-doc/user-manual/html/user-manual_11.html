2.2 SDF comment convention

The comment convention within an SDF specification is that character between "%%" and the end of line is comment as well as every character between two "%" including the newline character. An example of the use of comments is given below. 

module Layout

%% In this module we define the 
%% comment convention for Sdf.

exports
  lexical syntax
    "%%" ~[\n]* [\n]   -> LAYOUT
    "%" ~[\n\%]+ "%"   -> LAYOUT
SDF comment

This definition also defines the comment convention in SDF itself. More details on defining layout can be found in Restrictions.
