Term rewriting

Asf+Sdf specifications are executed as (conditional) rewrite rules. Both interpretation and compilation (using the ASF2C compiler [10]) of these rewrite rules are supported. The compiler generates very efficient C code that implements pattern matching and term traversal. The generated code uses ATerms as its main data representation, and ensures a minimal use of memory during normalization of terms.
