9.2 Medium Term (within 1.5 years)

  • Support for traversal functions in the compiler.
  • Support for origin tracking in evaluator and compiler.
  • New lexical primitives in ASF+SDF that allow the detailed control of the output of transformations.
  • User-interface tailoring: make it possible to add button definitions to editors that can activate either specified functions or external tools.
  • Generation of stand-alone environments.
  • Support for representing and manipulating graphs.
  • Introduction of generic services like error messaging and online help.
M.G.J. van den Brand and P. Klint, September 18, 2003
