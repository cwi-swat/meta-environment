9.3 Long Term (more than 1.5 years)

The long term ambition is to build a very flexible tool kit that can be used to build many kinds of interactive systems for program analysis and program transformation. We currently target domain-specific languages and software renovation as our main application areas. Emphasis will be on:

  • Enhancing the functionality, efficiency and reuasability of components.
  • Gradually defining a framework in which all these applications can be fit.
  • Carrying out case studies with as many different languages as possible to get feedback on the emerging framework.
M.G.J. van den Brand and P. Klint, September 18, 2003
