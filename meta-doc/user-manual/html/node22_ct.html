Parameterized Sorts

Sort names can have parameters. This provides a way of distinguishing a generic sort List for integers, e.g. List[[Int]], from booleans, e.g. List[[Bool]]. These sort parameters can be instantiated via the parameters of the module name. A parameterized sort may have the form List[[X,Y]] where X and Y are generic sorts which will be provided via the parameters of the module name. See Parameterization and Renamings[Section ]ParametersRenamings for more details. The context-free syntax rule describing parameterized sorts is: 

   Sort "[[" {Symbol ","}+ "]]" -> Symbol
