ToolBus Applications

This page contains an (incomplete) overview of the most important ToolBus applications so far. First we list the applications that are shipped with the standard ToolBus distribution.
  • The ToolBus viewer is the basic ToolBus debugger.
  • TIDE is an experimental ToolBus debugger that can be used to debug both scripts and tools.
  • There is an extensive set of demos located in the demos directory, including:
    • The auction demo: it implements a distributed auction.
    • The battleship demo: play the game of battleships against another player anywhere on the net.
    • The game of go-moku.
    • The hearts demo: play the game of hearts against other players and/or robot players. You can also program your own robot player!
    • Two versions of tic-tac-toe.
There are also a number of other applications being developed, most notably:
  • A simulator for embedded systems.
  • A muCrl simulator toolkit.
