The ToolBus - Software Components

The ToolBus implementation consists of the following software components:
  • The ToolBus interpreter (bin/toolbus)
  • The C tool interface generator (bin/ctif)
  • A number of adapters:
    • The gen-adapter
    • The bc-adapter
    • The perl-adapter
    • The python-adapter
    • The SWI-Prolog adapter
    • The wish-adapter
    • The tcltk-adapter
  • A number of demos:
    • The auction demo
    • The battleships demo
    • The calculator demo
    • The constraint demo
    • The edit demo
    • The game demo
    • The go-moku demo
    • The hearts demo
    • The hello demo
    • The perl demo
    • The tic-tac-toe demo
    • The SWI-Prolog tic-tac-toe demo
    • The wave demo
